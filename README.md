# Indoor Plant Expert System (AI-Based Recommendation)

This project is an **Expert System** developed as part of my **Master's degree in Intelligent Engineering Systems**. It simulates the decision-making process of a botanist to recommend indoor plants based on environmental conditions.

## 🚀 Key Features
- **Knowledge-Based Logic:** Uses a structured JSON database to store plant characteristics.
- **Weighted Matching Algorithm:** Instead of simple binary matching, the system uses a scoring mechanism to find the best possible match.
- **User Input Validation:** Ensures robust interaction through a validated command-line interface.
- **Match Percentage:** Provides users with a confidence score for each recommendation.

## 🛠️ Technologies Used
- **Language:** Python 3.14
- **Data Format:** JSON (for the knowledge base)
- **Concepts:** Rule-based Reasoning, Data Parsing, Object-Oriented Programming (OOP).

## 📂 Project Structure
- `expert_system.py`: The core logic and recommendation engine.
- `plants.json`: The database containing characteristics of 20+ indoor plants.
- `README.md`: Project documentation.

## 📖 How to Run
1. Ensure Python 3.x is installed.
2. Clone this repository.
3. Run `python expert_system.py` in your terminal.
